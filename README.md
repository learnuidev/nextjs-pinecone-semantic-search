# Next 14 Pinecone Semantic Search Example

## Prequesites

Create .env file and populate

```
PINECONE_API_KEY=
OPENAI_API_KEY=
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
